# enrollment-converter

입과등록양식 변환 자동화 도구입니다. GitHub Pages로 바로 배포할 수 있도록 정적 웹앱 형태로 정리되어 있습니다.

## 사용 방법

1. 저장소의 `docs/` 폴더를 유지한 채 `main` 브랜치에 올립니다.
2. GitHub 저장소 설정에서 `Pages`를 엽니다.
3. Source를 `Deploy from a branch`로 선택합니다.
4. Branch는 `main`, Folder는 `/docs`를 선택합니다.
5. 저장 후 몇 분 기다리면 사이트 주소가 생성됩니다.

## 배포 후 주소 예시

```text
https://사용자명.github.io/저장소명/
```

## 포함 파일

- `docs/index.html`
- `docs/app.mjs`
- `docs/engine.mjs`
- `docs/data.mjs`
