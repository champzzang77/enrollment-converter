# 입과 업로드 변환기

브라우저에서 엑셀 파일을 바로 읽어 `lms 일괄등록양식`으로 변환하는 정적 웹앱입니다.

## 특징

- 서버 없이 브라우저 안에서 직접 변환합니다.
- 업로드한 파일이 별도 백엔드로 전송되지 않습니다.
- 엑셀 라이브러리는 사이트 안에 직접 포함되어 외부 CDN 의존성을 줄였습니다.
- 최근 사용 기록은 각 사용자 브라우저에만 저장되며, 사용자명과 파일명은 남기지 않습니다.

## 배포

이 폴더는 GitHub Pages의 `docs/` 폴더로 바로 배포할 수 있습니다.

예시 주소:

```text
https://사용자명.github.io/저장소명/
```

## 로컬 전용 사용

인터넷 없이 내 PC에서만 실행하려면 아래 실행 파일을 사용하면 됩니다.

```text
/Users/parkchamin/vibe/start_local_enrollment_app.command
```

이 실행 파일은 로컬 서버를 켜고 브라우저에서 `http://127.0.0.1:8093` 주소를 자동으로 엽니다.

## 주의

- 중앙 로그는 없습니다.
- 다른 PC에서는 각자 자기 브라우저 기록만 보입니다.
- 새로운 파일 유형이 생기면 `data.mjs`에 프로필을 추가해야 합니다.
